Platform Roles

Three roles, one connected platform

Each role has precise access controls enforced at the middleware level. Mentors teach, parents manage, students learn.

🎓
Mentor

Subject experts who create and deliver lessons, schedule sessions, and track attendance.

  • Create & manage lessons
  • Schedule sessions with dates & topics
  • Write session summaries
  • Use AI-powered summarization
👨‍👩‍👧
Parent

Parents register, create student profiles, and book their children into lessons.

  • Create & manage student profiles
  • Browse all available lessons
  • Book students into lessons
  • Enroll students in sessions
📚
Student

Managed by parents. Students are created via the API and attend booked sessions.

  • Created by parent accounts
  • Attend booked lesson sessions
  • Tracked per-session attendance
  • Linked to a single parent

Clean, predictable endpoints

RESTful design with consistent JSON responses and role-based access guards on every route.

GET/meCurrent user
